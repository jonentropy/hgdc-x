Hgdc-X

GUI client for the Hackathon Gunther Daemon

About

Hgdc-X is a cross-platform desktop client for the awesome Hackathon Gunther Daemon music player project.

It is a feature complete GUI client, including SSL encryption, written in Lazarus, and is designed to target Linux (GTK2), Mac OS X (Carbon) and Windows (Win32). Other targets and widget sets, for example FreeBSD are possible but untested.

It will build with no additional libraries (except for synapse, which is statically linked and included in the source tree), with Lazarus 0.9.30 or later.

Thanks to the hgd developers for such a great music daemon, and hence opportunity for this client to exist.

Thanks to Hannah for "GUI consultancy", Shaun for Debian/Ubuntu packaging, Ed for Mac OS X testing and Omer for help with this site.

The logo was designed by Lucy Lindsell of theartofindigo.com and is licensed under CC BY-NC-ND.

Features

  • Fully compatible with the current release of the hgd server.
  • Implements hgd administrative commands.
  • Supports SSL encryption.
  • Album art display from Last.fm.
  • Cross-platform and native code.
  • Written in Lazarus/Freepascal.
  • Released under a liberal ISC open source licence.
